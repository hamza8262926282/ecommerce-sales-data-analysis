# E-Commerce Sales Data Analysis (Python + Power BI)

This project focuses on analyzing e-commerce sales data to uncover business insights and visualize sales performance using **Python and Microsoft Power BI**.

The project demonstrates a complete **data analytics workflow**, including data cleaning, exploratory data analysis (EDA), and interactive dashboard creation.

---

## Project Objectives

- Analyze sales performance across different categories
- Identify top-performing products
- Understand customer purchasing behavior
- Track sales trends over time
- Create an interactive Power BI dashboard for business insights

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Microsoft Power BI
- Jupyter Notebook

---

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Sales Trend Analysis
4. Product and Category Performance Analysis
5. Dashboard Development in Power BI

---

## Power BI Dashboard

The interactive Power BI dashboard provides insights such as:

- Total Sales and Revenue
- Product Category Performance
- Sales Trends Over Time
- Regional Sales Insights

---

## Project Files

- **Python Notebook:** Data cleaning and analysis
- **Power BI Dashboard:** Interactive visualization
- **Dataset:** E-commerce sales data
